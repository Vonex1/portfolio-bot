# Portfolio Telegram Bot

Этот бот - автоматизированный инструмент для презентации услуг и сбора заявок. Он разработан для обеспечения стабильной связи 24/7 и оптимизирован под требования современного бизнеса.

## Основные функции
- **Автоматизированная визитка:** Мгновенный ответ на приветствие и предоставление информации об услугах.
- **Сбор заявок:** Удобный механизм получения контактов от клиентов прямо в Telegram.
- **Бесперебойная работа:** Бот развернут в облаке с использованием системы мониторинга доступности.

## Технологии
- **Язык:** Python (версия с низким потреблением ресурсов).
- **Хостинг:** Render (Web Services).
- **Мониторинг:** Настроенный Ping-сервис для исключения спящего режима.

## Установка и запуск
Проект полностью готов к деплою на облачные платформы. Все конфигурационные параметры хранятся в переменных окружения (Environment Variables).
